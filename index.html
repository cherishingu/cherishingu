How to establish your blog

###1.安装git，https://git-scm.com

###2.安装node，https://nodejs.org/en/
这里版本太低可能出错

###3.打开可（tao）爱（yan）的shell
执行 

$sudo npm install -g hexo

这个时间有点长，不要着急哦，成功后大概这样

####初始化hexo
在用户目录下创建名为hexo的文件夹(或建立你喜欢位置的文件夹)，cd到此目录下，执行下面的命令： 

$hexo init

成功后

####安装依赖

正如上图所提示的，部署前不要忘了安装依赖 

$npm install

####生成静态页面

执行下面的命令，生成静态页面 

$hexo generate

成功后

####开启服务

开启本地服务器。Ps：停止服务使用Ctrl+C。（这个用不到，最后配置完结束就行了） 

$hexo server

成功后

然后就可以本地查看效果啦
打开浏览器，输入网址localhost
surprise！

###好，下面我们部署酷炫的github，我们需要做两件事：新建repository和配置SSH Key
1.新建名称格式：your_user_name.github.io 的仓库
2.来给这个仓库添加SSH key，参考官方文档做这两部分 Generating a new SSH key and adding it to the ssh-agent
git官方帮助链接
需要打的代码很少的，这一步方便后面的设置，一定记住你的账号密码哦

好下面我们进入最重要的部分，小编琢磨这个用了一天。。。

###修改配置文件
打开_config.yml文件，在文件末尾修改为如下效果，Ps:yuweiguocn改为你自己的用户名

deploy:              
 type: git          
    repo: https://github.com/yuweiguocn/yuweiguocn.github.io.git
    branch: master

##记住这里一定要切换到英文自己手打！冒号后面要加一个空格。然后保存就可以了

###安装git部署器
要先安装依赖，部署之前都要依赖

$nmp install

然后生成静态页面 

$hexo generate  （如果没问题话，显示接收不会是0）

最后（两个小短横，前面有空格哦）

$npm install hexo-deployer-git --save

部署完 

$hexo generate

再

$hexo deploy

显示已git，大功告成:)
打开浏览器输入网址：http://yuweiguocn.github.io Ps:yuweiguocn改为你自己的用户名
或者看你的github仓库设置page里也有这个网址，手打不知道为什么有时候进不去

ps:错误解决方法以及如何写自己的第一个博客，还有莫名其妙消失的图片，宝宝睡醒了再说哈哈
